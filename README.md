# Age Calculator

A simple **Age Calculator** built using HTML, CSS, and JavaScript.  
It calculates your exact age in **years, months, and days** based on your date of birth.

## 🔹 Features

- Calculate age in **years, months, and days**
- Responsive and visually appealing **gradient background**
- Input validation (alerts if no date is entered)
- Easy-to-use interface with a **hover effect** on the button
